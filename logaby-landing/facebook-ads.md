## Ad Creative Guidelines

### Image/Video Recommendations:

**Option 1: iPhone Mockup**
- Show the iPhone screen with Siri interface
- Voice command text: "Hey Siri, tell Logaby 4oz bottle"
- Clean, simple background
- Use your brand colors

**Option 2: Parent + Baby Photo**
- Exhausted parent holding baby
- Text overlay: "Hands full? Just talk."
- Authentic, relatable imagery

**Option 3: Before/After**
- BEFORE: Parent fumbling with phone while holding baby
- AFTER: Parent relaxed, just speaking
- Split-screen comparison

**Option 4: Video Ad (15-30 seconds)**
- Scene 1: Parent struggling with phone at 3am
- Scene 2: Parent using voice command
- Scene 3: Data synced across devices
- End: "Join the waitlist" CTA

### Design Tips:
- Use high-contrast text (readable on mobile)
- Include your logo
- Show the app interface if possible
- Keep it simple and clean
- Mobile-first design (most users on mobile)

## Campaign Setup in Facebook Ads Manager

### Step 1: Create Campaign
1. Go to [Facebook Ads Manager](https://business.facebook.com/adsmanager)
2. Click **Create** â†’ **Campaign**
3. Choose objective: **Traffic** or **Conversions** (if pixel installed)
4. Name: "Logaby Waitlist - [Month Year]"

### Step 2: Ad Set
1. **Audience**: Choose one of the audiences above
2. **Placements**: Automatic (Facebook & Instagram)
3. **Budget**: $5-10/day
4. **Schedule**: Continuous or set end date

### Step 3: Create Ads
1. Upload creative (image or video)
2. Copy/paste ad copy from variations above
3. Add UTM parameters to landing page URL
4. Preview on mobile & desktop

### Step 4: Launch & Monitor
1. Review and publish
2. Monitor daily for first 3 days
3. Check metrics: CTR, CPC, conversions
4. Pause underperforming ads
5. Scale winning ads

---

## A/B Testing Strategy

**Week 1**: Test all 5 ad variations
- Run each ad with $5/day budget
- Monitor CTR and conversion rate
- Identify top 2 performers

**Week 2**: Scale winners
- Increase budget on top 2 ads to $10-15/day
- Pause bottom 3 performers
- Test new creative variations

**Week 3-4**: Optimize
- Test different audiences
- Test different creative (images vs. video)
- Refine ad copy based on comments/feedback

---

## Key Metrics to Track

- **CTR (Click-Through Rate)**: Aim for 1-3%
- **CPC (Cost Per Click)**: Aim for $0.50-$2.00
- **Conversion Rate**: Aim for 20-40% (visitors â†’ signups)
- **Cost Per Signup**: Aim for $2-$5
